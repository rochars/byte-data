Source:

Turn a byte buffer into what the bytes represent.

Name Type Description
buffer !Array.<number> | !Array.<string> | Uint8Array An array of bytes.

options Object The options. They are: - "signed": If the numbers are signed. Default is false (unsigned). - "float": True for floating point numbers. Default is false.
